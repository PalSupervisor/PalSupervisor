Capabilities

Everything You Need, Nothing You Don't

🖥️

Server Control

Start, stop, restart, and graceful-shutdown from the UI. Live FPS, frame time, and uptime. Scheduled restarts with custom warning messages. Auto-restart on crash.

📊

Live Dashboard

Color-coded ring gauges for FPS, frame time, player count, and uptime. CPU, memory, and disk bars with health thresholds. Online player list auto-refreshed every 5 seconds.

👥

Player Management

View all online and historical players. Inspect inventory, Pals, stats, and base. Kick, ban, unban in one click. Give items and Pals. Global item search across the server.

🗺️

Live Map

Real-time positions of players and guild bases. Loot browser with category filtering. Teleport any player anywhere on the map. Summon or teleport to players.

🏰

Guild Management

Browse every guild on the server. View members, levels, and bases. Drill into base chests and base Pals. Full guild oversight for moderators.

🛡️

Anti-Cheat

Scheduled automatic scans with configurable cadence. Fine-grained level-gap thresholds. Kick, ban, reset cheaters, wipe Pals, or delete illegal items. Per-flag moderation log.

🔧

Mod Manager

Upload mods with metadata. Enable, disable, or toggle in bulk. Drag-style load order editor. Dependency validation, update checks, and profile export/import.

🌐

Nexus Mods Integration

Browse Trending, Latest, and Recently Updated mods via Nexus API. Search the full catalog, view changelogs, and one-click install directly to your server.

💾

Backup System

On-demand and scheduled world snapshots. Configurable interval and retention. Backup-before-restart safety net. One-click restore of any snapshot.

📈

Monitoring & Logs

Real-time server stats with historical charts. Live alert feed. Filterable log viewer. API health dashboard for RCON, PST, and Game REST. Crash detection and reporting.

🔐

User & Permission System

Multi-user support with 6-tier role hierarchy. Fine-grained per-page, per-tab, and per-action permissions. JWT auth with auto-logout. Permission-aware sidebar and routes.

🎨

Themes & Polish

Multiple curated themes with customizable card styles, border glow, and animations. Animated particle backgrounds. Fully responsive layout with mobile-friendly sidebar.

Quick Start

Up and Running in Minutes

Prerequisites before installing:

⚙️ Palworld Dedicated Server 🔌 RCON Enabled 🌐 REST API Enabled 🛡️ PalDefender Installed
1

Download the latest release

Grab PalSupervisor.rar from the GitHub Releases page.

2

Extract the archive

Extract the .rar file to any folder on your server machine.

3

Run PalSupervisor.exe

The dashboard opens automatically at http://localhost:3000.

4

Complete the 5-step setup wizard

Point to your PalServer.exe, configure RCON & REST API, set up PST auth, and configure memory management. Done!

5

Get your license key

Free, trial, and paid keys available — join the Discord server to get yours.
